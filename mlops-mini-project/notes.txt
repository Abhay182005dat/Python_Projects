Gunicorn expects:
  <module_path>:<flask_object>
So:
  flask_app.app:app

means:
flask_app.app → Python module path
app → Flask instance inside that module

When you set: PYTHONPATH: flask_app ,Python will treat : mlops-mini-project/flask_app  as the root directory.
